Bu klasöre GUI sekme başlıklarında gösterilecek PNG görsellerini koyun.

Beklenen örnek dosya adları:
- turning.png
- milling.png
- material.png
- drilling.png

Notlar:
- Tkinter için PNG en sorunsuz formattır (Pillow bağımlılığı eklemeden).
- Dosya yoksa GUI otomatik olarak yer tutucu (placeholder) gösterir.
